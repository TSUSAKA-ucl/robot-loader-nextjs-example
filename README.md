This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

##  to get started

1. `pnpm create next-app@latest`
2. Select default options
3. `cd <project-directory>`
4. copy [`.npmrc`](./.npmrc) from this repository to your project directory
5. copy the followings to your `app` directory  
   * [`page.tsx`](./app/page.tsx)
   * [`App.jsx`](./app/App.jsx)
   * [`globals.css`](./app/globals.css)
6. add the following packages:
   ```
   pnpm add aframe
   pnpm add @ucl-nuee/robot-loader @ucl-nuee/ik-cd-worker
   pnpm add @ucl-nuee/ur5e
   ``
7. run the asset copy script to copy necessary assets to the public folder:
   ```
   npx copy-assets
   ```
8. build and run the development server:
   ```
   pnpm build
   pnpm dev
   ```
## Note

This directory is an example of modifying a package automatically
generated by `pnpm create`. Important files automatically generated,
such as `package.json`, are written in `.gitignore`, so `.gitignore`
in this directory cannot be used as is.
